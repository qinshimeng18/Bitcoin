##项目简介##
>1、对基于P2P网络的匿名电子货币-比特币的交易数据进行读取，有近50G的数据，每一条交易信息只有几百字节  
>2、按年份构建比特币交易复杂网络，计算得出网络的基本属性  
>3、对比特币的网络进行可视化  
>4、对泄漏的MTGOX交易网站泄漏的信息进行匹配，探索同一用户的交易模式  
>5、用真实的数据得出的特性去反推整个比特币交易网的特性，并实现去匿名化
